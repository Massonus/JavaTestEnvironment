Flexbox

Flexible Box Layout — модуль, що визначає особливий режим верстки, який називається flex layout. Завдяки Flexbox простіше створювати складні, комплексні інтерфейси, де ми з легкістю можемо перевизначати напрямок та вирівнювання елементів або створювати адаптивні табличні представлення. Основними складовими компонування flexbox є flex container та flex items. Flex container — це елемент, усередині якого розміщені flex-елементи.
